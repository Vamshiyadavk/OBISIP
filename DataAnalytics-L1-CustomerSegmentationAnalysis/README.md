# Customer Segmentation Analysis

This project performs customer segmentation on the Online Retail dataset using customer purchasing behavior. The goal is to group customers into meaningful segments that can help businesses understand buying patterns and create targeted marketing strategies.

## Project Objectives

- Clean and prepare retail transaction data
- Calculate customer-level purchasing metrics
- Analyze Recency, Frequency, and Monetary (RFM) behavior
- Use K-Means clustering to segment customers
- Visualize and interpret customer groups

## Dataset

The project uses the `Online Retail.xlsx` dataset, which contains transaction-level retail sales data.

## Technologies Used

- Python
- Pandas and NumPy
- Scikit-learn
- Matplotlib and Seaborn
- Jupyter Notebook
